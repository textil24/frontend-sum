# Стек:
- **Vite** 
- **Tailwind** 
- **Radpid API** 

# Запуск проекта:

### 1) Чтобы установить зависимости, выполните следующую команду:

`npm install`

### 2) Чтобы запустить процесс разработки, используйте следующую команду:

`npm run dev`
